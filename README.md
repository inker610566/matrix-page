Requirement
===========

* python
* staticjinja: http://staticjinja.readthedocs.org/en/latest/

How to Run
==========

1. git clone
2. cd to this repository directory

	staticjinja build
3. the result file will be "gen.htmld" in current directory

Parameter tuning
================
	
* all params are code in header of templates/gen.htmld
